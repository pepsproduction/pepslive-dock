## Logo Sources

ใช้เป็น OBS Image Source เท่านั้น

```text
PEPS_LogoA
PEPS_LogoB
```

ถ้า source เดิมเคยเป็น Browser Source ให้กด Settings > Logos > ซ่อม Logo Image Sources

## Media / FX Sources

```text
PEPS_TeamA_FX
PEPS_TeamB_FX
```

## วิธีสร้าง

ไปที่ Settings > OBS แล้วกดปุ่ม `+` ในแต่ละ Tag หรือกด `สร้างทั้งหมด`
